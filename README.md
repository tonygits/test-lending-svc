Getting Started
===============

- Clone repo
- Install Golang 1.20 [Download](https://golang.org/dl/#go1.20.2)
- Install Docker [Download](https://docs.docker.com/)
- Run Tests: `make test`
- RUN microservice: `make build_run`
- Stop && remove svc containers and remove image: `make down`
